# Security policy

Do not open a public issue for a suspected vulnerability. Report it privately
through GitHub Security Advisories for `faustbrian/keyphrase` and include the
affected version or commit, reproduction, impact, and proposed mitigation when
available.

Generated values, mnemonics, entropy, passphrases, seeds, private keys, and
production randomness traces must not be included in reports. Replace them with
deterministic public fixtures.

Stable releases receive security fixes. Pre-release commits may change public
APIs while the independent design review remains pending.
