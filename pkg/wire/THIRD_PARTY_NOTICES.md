# Third-Party Notices

## Policy

This file is mandatory even when no copied or vendored code requires a notice.
It records attribution for copied, forked, generated, or vendored source.
Ordinary Go module dependencies retain their upstream licenses and notices.

## Current Attribution

No third-party source code is copied or vendored in this repository. TOML,
YAML, MessagePack, CBOR, and BSON support use the direct Go module dependencies
listed in `go.mod`; those projects retain their upstream licenses and notices.
See [docs/dependencies.md](docs/dependencies.md) for the maintained dependency
and interoperability rationale.
