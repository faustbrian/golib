# Roadmap

## Delivered in v1.0.0

- PostgreSQL 14-18 hosted integration evidence
- exact meaningful production coverage and green release gates
- API, security, operational, adoption, and compatibility review
- stable API publication after hosted CI confirmed the tagged commit

## After v1

Future work may add narrowly scoped pgx hooks or observability instruments.
The project will not become an ORM, query builder, migration engine, repository
generator, transparent retry executor, or generic database abstraction.
