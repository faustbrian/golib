# Roadmap

## Before v1

- Keep the completed local hardening matrix and evidence current.
- Validate hosted workflows after the local work is finished.
- Publish v1 only when API/wire compatibility, PostgreSQL matrices, and all
  acceptance criteria are green.

## Possible later adapters

- Lossless schema.org `OpeningHoursSpecification` mappings.
- Optional localized labels through `localized`.

Provider prose parsing, appointments, workforce planning, payroll, cron,
holiday datasets, HTTP transport, and business eligibility remain non-goals.
